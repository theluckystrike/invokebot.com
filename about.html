About InvokeBot

InvokeBot is a free, browser-based HTTP request builder designed for developers who work with webhooks, APIs, and microservices. It provides a visual interface for constructing requests, sending them, and generating equivalent code in multiple languages.

About This Project

InvokeBot is a free, browser-based API testing tool. Paste a URL, set headers, and inspect the response — no accounts or installs needed.

  • Lochbot — Chatbot response tester
  • Kappafy — JSON explorer and mock API generator

Privacy

InvokeBot runs entirely in your browser. Request history is stored in localStorage on your device. The only network requests made are the ones you explicitly send using the tool.
